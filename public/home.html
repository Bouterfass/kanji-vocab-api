Japanese vocabulary API

What is it?

This API allows you to retrieve all the vocabulary you need to start learning japanese.
It provides over 1500 words classified by their JLPT (Japanese Language Proficiency Test) levels.

Endpoints

    Here are the different ways to reach the vocabulary:
  • by level: the JLPT's levels goes from 5, the lowest, to 1, the highest (more commonly called N5 to N1), but it appears that there aren't new words to learn at the N1 level.
  • by type of word: Noun, Adverbial-Noun, Verb, Ichidan-verb, Transitive-verb, Pre-noun-adjectival, Adverb, Godan-verb, Intransitive-verb, Adjective, na-adjective, i-adjective, Suffix, Expression, Prefix, Adverbial-noun, Pronoun, Temporal-noun, Katakana, Numeric, Conjunction, Wasei.
